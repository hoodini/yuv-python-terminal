## How to Take Screenshots:

### On Windows:
- Press `Win + Shift + S` for Snipping Tool
- Or use `PrtScn` key and paste into Paint

### On Mac:
- Press `Cmd + Shift + 4` for selection tool
- Press `Cmd + Shift + 3` for full screen

### Using Browser DevTools:
1. Open DevTools (F12)
2. Toggle device toolbar (Ctrl+Shift+M)
3. Select device or custom size
4. Take screenshot

### Best Practices:
- Use high resolution (at least 1920px wide for desktop)
- Keep file sizes reasonable (use PNG format)
- Ensure text is readable
- Show the app in action, not just idle
- Capture the animations if possible (use screen recording then extract frame)

## After Taking Screenshots:
